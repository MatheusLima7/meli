# Meli - server

## Estrutura do projeto

```
    - @constants                       -> Pasta que contém as principais constantes do Projeto
        - constants.js
        ...
    - @domain-services
        - parsers                      -> Pasta que contém os parsers dos responses e services
          {name-parser}.js             -> Nome referente ao response (- pasers)
        {name}                         -> Nome do service
        ...
    - @settings                        -> Pasta com as configurações
        ...
    - @respositories                   -> Pasta com os requests
```
